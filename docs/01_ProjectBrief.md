# 🎯 프로젝트 개요

**리오시스템_v1.0 (RIO System v1.0)**  
리오 개인의 실무 환경에서 상세페이지 제작 전 과정을 자동화하기 위한 **로컬 전용 경량 시스템**이다.  
기존 AI 상세페이지 자동 생성 시스템 2.0을 기반으로 불필요한 협업·배포 기능을 제거하고  
ComfyUI 기반의 **로컬 이미지 변환 엔진**을 중심으로 재구성되었다.

---

## 1. 시스템 철학

> “반복은 자동화하고, 감각은 보존한다.”

리오시스템은 AI가 리오의 작업 습관·문체·톤앤매너를 그대로 재현하여  
매일 반복되는 상세페이지 제작 업무를 1/5 이하의 시간으로 단축시키는 것을 목표로 한다.

---

## 2. 구축 목적

| 구분 | 내용 |
|------|------|
| **핵심 목적** | 상세페이지 제작 프로세스를 완전 자동화 |
| **대상 사용자** | 리오 (개인 실무 중심) |
| **적용 범위** | 농수산물·식품 브랜드 상세페이지 / 스마트스토어·자사몰용 콘텐츠 |
| **비적용 범위** | 외주용 멀티유저 구조 / SaaS 배포형 기능 |
| **핵심 가치** | “AI가 리오의 감각으로 일한다.” |

---

## 3. 주요 목표

1. **텍스트 자동화** – GPT가 리오 문체에 맞게 상품 스토리 생성  
2. **이미지 자동화** – ComfyUI가 품종·연출 기준으로 시각적 결과물 생성  
3. **디자인 자동배치** – Figma 플러그인이 텍스트·이미지를 자동 배치  
4. **QA 자동검증** – 문체, 해상도, 색감 일관성 자동 점검  
5. **완전 단방향 구조** – Sheets → GPT → ComfyUI → Figma → Export

---

## 4. 운영 환경

- 로컬 실행 중심 (ComfyUI)
- GPT API, Google Sheets API, Figma API 연동
- VSCode 개발 환경 / TypeScript 기반
- 데이터는 `/dist` 및 `/logs` 폴더 내 단계별 캐싱 관리

---

## 5. 결과 기대치

| 항목 | 기준 | 개선 목표 |
|------|------|------------|
| 평균 상세페이지 제작 시간 | 약 4~6시간 | **1시간 이내로 단축** |
| 품질 편차 | 작업자 감각 의존 | **AI 규칙 기반 통제** |
| 반복률 | 매일 유사 업무 반복 | **자동화율 80% 이상** |
